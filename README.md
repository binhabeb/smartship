# SmartShip Logistics Platform

SmartShip is a modern, high-performance logistics management platform built with Next.js and Supabase.

## Features
- **Public & Private Portals:** Customer tracking and request forms alongside a secure Admin Dashboard.
- **Dynamic Tracking:** Real-time shipment status updates.
- **Responsive Design:** Optimized for both desktop and mobile experiences.
- **Supabase Integration:** Secure database and authentication.
- **Vercel Ready:** Fully optimized for seamless deployment.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment
This project is configured for optimal deployment on [Vercel](https://vercel.com). The `next.config.ts` includes configurations for Image Optimization, Compression, and strict security headers.
